NilFinder

public class NilFinder<VC: UIViewController, C>: Finder

Dummy class to be provided to an assembly to show that this that never should be found in a view controller stack and should always be created from scratch. The only purpose it exist is to provide type safety checks for ScreenStepAssembly.

For example UIViewController of this step was already loaded and integrated in to a stack by a storyboard.
